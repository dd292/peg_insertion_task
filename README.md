To install the environment use

`conda env create -f environment.yml`
`conda activate peg_insertion_env`

`python teleop.py` to run the data collection
use controls 
```
Move: W/A/S/D (XY), T/G (Z)
Rotate (body): Z/X=Roll, C/V=Pitch, B/N=Yaw
Grip: [ / ]  Reset: R   Quit: Esc
```
`python simple_sim.py` to run the simple sim
